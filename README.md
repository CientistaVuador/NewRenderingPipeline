# NewRenderingPipeline
Creating and testing a new rendering pipeline

Java 17 and OpenGL 3.3 is required

Controls:

WASD - Move
Space - Jump
F - Flashlight
V - Noclip

Bake:
F4 - Bake lightmaps
F5 - Bake cubemaps

Debug:
F1 - Toggle reflections
F2 - Toggle parallax mapping
F3 - Show reflections only
F6 - Show ambient cubes
